HOW PLAN SYMBOL RECOGNITION WORKS
---------------------------------

Fast Vision:
- one AI vision pass;
- quicker and lower cost;
- reads the drawing legend, symbols, schedules and visible annotations;
- more risk, so every editable quantity must be checked.

Smart Review:
- deeper AI review of relevant pages, legends, rooms, notes and revisions;
- checks duplicate counts more carefully;
- slower and lower risk, but still requires a Builder/licensed-trade check.

Both modes work for Electrical, Plumbing and Cladding and use the existing AC
catalogue ordering and rates. Carpentry is only a hidden disabled code scaffold
until a verified catalogue and rules are added later.

Important limitation: one final plan cannot prove that an item was moved or
replaced. The system classifies relocation/replacement only when existing,
demolition and proposed plans, revision notes or explicit annotations support it.

For graphical Word plans, exporting Word to PDF before upload gives the most
reliable symbol recognition. Word/TXT/CSV are accepted, but PDF preserves the
page geometry and embedded drawing best.


PRICING
-------

Plan customer estimate:
1. AC catalogue subtotal ex GST
2. add 20% profit
3. add 10% GST to that result

Example: $1,000 + 20% = $1,200; GST = $120; total = $1,320.

Quote Analysis compares each quoted ex-GST line with the matching AC ex-GST
catalogue line:
- above by more than $100 = EXPENSIVE
- below by more than $100 = CHEAP
- from -$100 through +$100 = FAIR
- no reliable match = NEED REVIEW


WHERE PROJECTS ARE SAVED
------------------------

Projects, tasks and result metadata are saved in this browser's localStorage.
Attachments are saved in this browser's IndexedDB. They are on the current
browser/device only — not in GitHub, not on the website server, and not in
Supabase. They do not automatically sync between devices.

Private/incognito storage can disappear when private tabs are closed. Use the
Projects page > Export Backup regularly. Restore Backup can load that JSON file
on another browser/device.
