VICTORIA PERMIT CHECKLIST — 15 JULY 2026
==========================================

- Added a rule-based Permit Checklist for Victorian building and renovation
  projects without using AI.
- Project-type and optional scope questions flag possible planning, building,
  report-and-consent, protection work, owner-builder, trade, pool, demolition,
  asbestos, council-asset, easement and overlay requirements.
- Results deliberately use only “Likely required” or “Confirm with authority”;
  the tool never makes a final “no permit required” decision.
- Added a project-specific document checklist with completion ticks and notes.
- Added prominent warnings that the tool can be incomplete or wrong and that
  council, a registered building surveyor and relevant authorities must confirm
  the result in writing before work starts.
- Strengthened every Permit Checklist warning to call the result a best-effort
  guess that is not always true, cannot prove a permit is or is not required,
  and must be checked with the correct qualified person or authority.
- Rechecked the rules against current official Victorian sources on 15 July
  2026 and display that review date with each generated result.
- Reworded the final result notice in a calmer, more professional tone while
  preserving the requirement to obtain written confirmation before proceeding.
- Added official Victorian source links, Save Draft, Print, Save to Project,
  reopen/edit support and automatic Schedule follow-up tasks.
- Connected the tool to dashboard search, Projects, sign-in protection,
  privacy-safe usage analytics and the offline PWA cache.

VICTORIA PROPERTY VALUE GUIDE — 14 JULY 2026
=============================================

- Added a guided residential Property Value Guide for Victoria.
- Kept only suburb, region and property type as essential questions; all other
  answers are optional and improve the evidence-confidence score.
- Uses recent comparable sales first, a matching suburb median second and the
  December 2025 Valuer-General Victoria broad market figures only as fallback.
- Adjusts comparable sales transparently for sale timing, land/floor area,
  bedrooms, bathrooms, car spaces and condition when those facts are supplied.
- Shows a midpoint, defensible range, evidence source, confidence score, every
  applied adjustment and the missing information that would improve the guide.
- Added plan/photo/PDF attachments for human review without using AI.
- Added notes, follow-up date, Save Draft, Print / Save PDF, Save to Project,
  reopen/edit support and automatic project follow-up tasks.
- Added multi-file Project attachments while retaining old single-file records.
- Added the tool to dashboard search, Projects quick links and offline PWA cache.
- Clearly distinguishes a preliminary guide from a formal valuation, agent
  estimated selling price and Victorian Statement of Information.
